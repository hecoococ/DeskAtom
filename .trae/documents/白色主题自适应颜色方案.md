# 白色主题自适应颜色方案

## 问题描述

当主色为白色或接近白色时，以下 UI 元素可能看不见：

* 标题栏的标题文字、图标、按钮

* 任务统计的标题和数值

* 筛选器的按钮文字

* 添加任务按钮的加号图标

* 其他使用白色或主题色的元素

## 技术方案

### 1. 颜色检测算法

使用 YIQ 颜色空间亮度计算：

```javascript
// 检测颜色是否接近白色（亮度高）
const isLightColor = (r, g, b) => {
  // YIQ 公式计算感知亮度
  const yiq = ((r * 299) + (g * 587) + (b * 114)) / 1000
  // yiq >= 128 为亮色，>= 200 为接近白色
  return yiq >= 200
}
```

### 2. 实现方式

#### 方案 A：CSS 变量（推荐）

在 App.vue 中添加新的 CSS 变量：

```javascript
'--text-on-theme': isLightColor ? '#1f2937' : '#ffffff'
```

#### 方案 B：动态类名

根据主题色亮度添加类名：

```html
<div :class="{ 'light-theme': isLightTheme }">
```

### 3. 推荐方案：CSS 变量

**优点：**

* 统一管理，一处修改全局生效

* 不需要修改大量 HTML

* 性能好，利用 CSS 变量的层叠特性

## 实施步骤

### 步骤 1：在 App.vue 添加颜色检测和 CSS 变量

```javascript
// appStyle computed 中添加
const p = appSettings.primaryColor
const yiq = ((p.r * 299) + (p.g * 587) + (p.b * 114)) / 1000
const isLightTheme = yiq >= 200

return {
  // ... 其他变量
  '--text-on-theme': isLightTheme ? '#1f2937' : '#ffffff',
  '--text-on-theme-secondary': isLightTheme ? '#4b5563' : 'rgba(255, 255, 255, 0.9)'
}
```

### 步骤 2：修改 TitleBar.vue

将硬编码的 `white` 替换为 `var(--text-on-theme)`：

```css
.logo-icon { color: var(--text-on-theme); }
.title-text { color: var(--text-on-theme); }
.badge { color: var(--text-on-theme); }
.control-btn { color: var(--text-on-theme); }
```

### 步骤 3：修改 App.vue 相关元素

```css
.add-btn { color: var(--text-on-theme); }
.filter-btn.active { color: var(--text-on-theme); }
.stat-value { color: var(--theme-color); } /* 保持主题色，但需确保对比度 */
```

### 步骤 4：处理边界情况

* 双色模式：使用主色的亮度判断

* 暗夜模式：已有独立处理，不受影响

* 渐变背景：使用渐变起始色的亮度

## 测试用例

1. **纯白色主题** (r:255, g:255, b:255)

   * 标题栏文字应为深灰色

   * 图标应为深灰色

   * 按钮应为深灰色

2. **浅色主题** (r:240, g:240, b:240)

   * 同上

3. **深色主题** (r:50, g:50, b:50)

   * 保持白色文字

4. **橙色主题** (r:255, g:140, b:0)

   * 保持白色文字

5. **双色模式**

   * 使用主色判断亮度
