# 主题颜色改进计划

## 需求概述
将主题颜色设置改进为支持"单色"和"双色"两种模式，通过按钮切换，并根据选择动态调整色板数量和布局。

## 功能设计

### 1. 模式切换按钮
- 在设置面板添加两个切换按钮："单色"和"双色"
- 使用分段控制器样式（Segmented Control）
- 当前选中的模式高亮显示

### 2. 单色模式
- 只显示一个色板
- 用于选择单一主题色
- 应用于：
  - 主界面毛玻璃效果的边框和背景色调
  - 按钮、图标等UI元素

### 3. 双色模式
- 显示两个色板（主色 + 辅色）
- 色板顶部添加配色方案选择栏
- 将毛玻璃的6个配色方案移动到顶部作为快捷选择
- 应用于：
  - 专注模式的渐变背景
  - 其他需要渐变效果的场景

## 实施步骤

### 阶段一：数据结构和状态管理
1. **修改 App.vue**
   - 添加 `themeColorMode` 属性（'single' | 'dual'）
   - 添加 `primaryColor` 和 `secondaryColor` 属性
   - 更新 `applySettings` 函数处理新属性
   - 更新 CSS 变量绑定逻辑

2. **修改 SettingsPanel.vue**
   - 更新 `defaultSettings` 添加新属性
   - 更新 `settings` reactive 对象
   - 更新 `loadSettings` 和 `saveSettings`

### 阶段二：设置面板UI改造
3. **添加模式切换按钮**
   - 创建分段控制器组件样式
   - 实现单色/双色切换逻辑

4. **改造色板区域**
   - 单色模式：显示单个 ColorPicker
   - 双色模式：
     - 顶部显示配色方案快捷选择（6个色块）
     - 下方显示两个 ColorPicker（主色 + 辅色）
     - 点击配色方案自动填充两个色板

### 阶段三：样式和应用逻辑
5. **更新毛玻璃效果**
   - 单色模式：使用 `themeColor` 作为边框和背景色调
   - 双色模式：使用 `primaryColor` 和 `secondaryColor` 生成渐变

6. **更新专注模式**
   - 根据模式选择使用单色或双色渐变

### 阶段四：持久化和兼容性
7. **保存/加载设置**
   - 确保新属性正确保存到 localStorage
   - 向后兼容（旧数据默认使用单色模式）

## 注意事项
1. 双色模式下，配色方案快捷选择应实时更新两个色板的值
2. 切换模式时，应保留当前颜色值（不重置）
3. 暗夜模式下配色方案选择栏的样式需要适配
4. 确保专注模式能正确接收和使用新的颜色设置
